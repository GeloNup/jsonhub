# 🚀 JSONHub - API Explorer



**JSONHub** es una aplicación web moderna diseñada para la exploración y visualización dinámica de datos provenientes de microservicios. Este proyecto fue desarrollado para la clase de JavaScript, enfocándose en la integración de APIs REST, manejo de asincronía y diseño de interfaces responsivas.

---

## 🛠️ Tecnologías Utilizadas

*   **JavaScript (ES6+):** Lógica central y manipulación dinámica del DOM.
*   **Vite:** Herramienta de construcción rápida para el desarrollo frontend.
*   **Bootstrap 5:** Framework de diseño para una interfaz moderna y adaptativa.
*   **JSONPlaceholder:** Fuente de datos (API REST) para las pruebas de integración.
*   **Google Fonts & Icons:** Tipografía *Inter* y *Bootstrap Icons*.

---

## 📋 Características Principales

1.  **Feed de Publicaciones (Posts):** Visualización de contenido con sistema de paginado dinámico.
2.  **Directorio de Usuarios (Users):** Tarjetas de contacto detalladas con información de perfiles.
3.  **Gestión de Registros (Todos):** Listado técnico de registros con identificadores únicos.
4.  **Diseño Glassmorphism:** Interfaz limpia con degradados, sombras suaves y efectos de elevación (*hover*).
5.  **Navegación Fluida:** Sistema de rutas modulares y botones de retorno rápido.

---

## 🚀 Instalación y Uso

Para ejecutar este proyecto de forma local, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/GeloNup/jsonhub.git](https://github.com/GeloNup/jsonhub.git)
    ```
2.  **Entrar a la carpeta:**
    ```bash
    cd jsonhub
    ```
3.  **Instalar dependencias:**
    ```bash
    npm install
    ```
4.  **Iniciar servidor de desarrollo:**
    ```bash
    npm run dev
    ```
5.  Abre en el navegador la URL que muestra Vite (normalmente `http://localhost:5173`).
